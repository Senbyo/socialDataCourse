  1. Looking at Fig 3-1, Janert writes "the data itself shows clearly that the amount of random noise in the data is small". What do you think his argument is?

    2. The argument is that the relationship between x and y is clearly replicated throughout the data, and does not vary a lot.

  2. Can you think of a real-world example of a multivariate relationship like the one in Fig 3-3 (lower right panel)?

    3. The curve look very much like a plot of the test and training error. Where the training error of a given stistical model goes towards zero, but as the model starts to overfit the test error starts to increase.

  3. What are the two methods Janert metions for smoothing noisy data? Can you think of other ones?

    4. Janert mentions weighted spiles and locally weighted regression(LOWESS). There are others, for example ordinary least squares or K nearest neighbours, and radial basis functions. ( Janert says that OLS is very much misused, so maybe we should remove it from this list?)

  4. What are residuals? Why is it a good idea to plot the residuals of your fit?

    5. Residuals is the distance between the estimated y values and the actual y values. In other words it is a measurement for how well the estimated y values actually fit to the underlying data.

  5. Explain in your own words the point of the smooth tube in figure 3-7.

    6. THe point of the smooth tube in figure 3-7 is to give an interval for which most of the data lies within. This can be seen as a confidence interval for the approximation. Furthermore it can be used to indentify regions that could potentially have outliners. This can be observed through width of the tube expanding.

  6. What kind of relationships will a semi-log plot help you discover?

    7. They will help deiscover exponential and power law behavior. It can help you plot the relative change of the y value rather than the absolute.

  7. What kind of functions will loglog plots help you see?

    8. THey make it possible to visualize data that is spanning many orders of magnituide, as well as revealing power law relationships.

  8. What the h#ll is banking and what element from our visual system does it use to help us see patterns? What are potential problems with banking?

    9. Baking is controlling the slope of the curve that is being plotted. With banking one aims to create a 45 degree slope on the curve.. This is done because it is much harder to see the change in a curve that is nearly vertical and horiziontal. THis is done by changing the aspect ration between the y and x-axis. A potential problem with baking is that one could lose to much detail by shrinking one of the axes a lot, which in turn can make it hard to compare each extreme of the axis that has not been shrunk. Furthermore it does make the reading of the plot harder because of the crooked aspect ratio of the two axis.

  9. I think figure 3-14 makes an important point about linear fits that is rarely made. What is it?

    10. The point here is that linear regression is not something that should be used to find a best fit of the plotted data. It should be used to create a predictor based on the data. When doing graphs and visualizing data it is about visualizing and summarizing the data and trends in the data and not create a model that predicts the data well.

  10. Summarize the discussion of Graphical Analysis and Presentation Graphics on pp. 68-69 in your own words.

    11. In short there are two different approaches to graphical visualization. The first one is graphical Analysis. In this case one wants to investigate and explore that data with the purpose of finding new information about the data. This could be considered exploratory data analysis. When graphical analysis is appropriate the approach to the data should be to create many graphics, varying as much as possible in order to view the data from as many different angles as possible.
    In the case of graphical presentation one strives to visualize information about the data that has already been invesitated and understood. This could be considered explanatory data analysis. For graphical presentation very polished graphs that are easy to understand with appropiate embellishment is the goal.
    This distinction is very important when it comes to creating plots and using 3D.js mainly because a more time consuming plot quickly makes one reluctant to start over, and as such it might make sense to do quick visualizations in other software before the graphical presentation is created in 3d.js.
