  2. What's the difference between the class and the id HTML attributes.

    3. The difference lies in the fact that there ids are unique for each DOM/HTML document. Meaning that only one element can have a given ID. Furthermore IDs cannot start with numerals. An element can only have one id Classes on the other hand can be used to class different elements together. And elements can be part of several classes. Meaning that IDs are unique and classes are not.

  3. What does "DOM" stand for? Explain in your own word what it refers to.

    4. DOM stands for Document Object Model. it referes to the hierarchy of a HTML page. Further more it defines the relationship of different elements within this HTML page. This relationship is defined through the use of parent, child, sibling etc.
